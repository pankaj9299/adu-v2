# ADU Container Product Configurator

A multi-step dynamic product configurator enabling users to customize modular ADU 
(Accessory Dwelling Unit) containers in real time, with complex conditional UI state 
managed across configuration steps.

🔗 [Live Demo](http://adu-container.com) 

## 🛠️ Tech Stack
React.js · Vite · Redux Toolkit · Tailwind CSS · REST API

## ✨ Features
- Multi-step configurator with dynamic conditional rendering
- Real-time UI updates based on user selections
- State persistence across steps via Redux Toolkit
- Fully responsive design

## 🚀 Getting Started

```bash
git clone https://github.com/pankaj9299/adu-v2.git
cd adu-v2
npm install
npm run dev
```
